### 项目使用技术栈

本项目使用到的技术栈如下：
* Vue全家桶
* Typescript
* Element Plus

## 加入项目

如果想加入本项目的开发，需将项目fork到自己的仓库下，功能开发完成后，提PR即可。

### 项目运行

启动项目

```
yarn install
```

```
yarn serve
```

项目打包

```
yarn build
```
